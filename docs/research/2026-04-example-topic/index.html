예시 리서치 주제

사이트 동작 확인을 위한 샘플 리서치입니다. 단일 페이지 + 커스텀 에셋 + 다운로드 첨부 패턴을 한 화면에 보여줍니다.

배경

이 페이지는 실제 리서치가 아니라 사이트 골격이 정상 동작하는지 확인하기 위한 예시입니다. 실제 리서치를 추가할 때 이 폴더 구조를 그대로 참고하면 됩니다.

Markdown 원본은 같은 폴더의 notes/에 두고, 화면용 HTML은 이 index.html에 작성합니다. 문서가 여러 개일 경우 notes/01-background.md, notes/02-findings.md 식으로 나눠서 보관할 수 있습니다.

커스텀 에셋 예시

아래 강조 박스는 이 리서치 폴더의 assets/style.css에서만 정의된 스타일입니다. 공통 CSS 다음에 로드되기 때문에 토큰을 그대로 활용하면서도 이 페이지에서만 보이는 요소를 만들 수 있습니다.

이 박스는 .r-example__callout 클래스로 스타일됩니다. 다른 리서치 페이지에서는 보이지 않습니다.

다중 페이지가 필요할 때

리서치가 여러 화면으로 나뉘어야 한다면 같은 폴더 안에 pages/01-background.html, pages/02-method.html 같이 추가하고, 각 페이지에서 사이드바 partial을 공유하면 됩니다. 해시 라우팅이나 SPA 방식은 사용하지 않습니다 - 각 페이지가 개별 정적 파일이어서 직접 링크 공유와 새로고침이 안전합니다.
