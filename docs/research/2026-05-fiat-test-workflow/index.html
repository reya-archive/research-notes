무엇인가

규칙을 문서로만 두지 않습니다

CLAUDE.md 한 장으로는 LLM이 매번 규칙을 지키기 어렵습니다. 이 워크스페이스는 규칙을 실행 가능한 자산으로 쪼개어, 작업 시점마다 자동으로 개입합니다.

룰북이 길잡이

도메인 무관 결정 트리와 테스트 11축 기준을 SSOT로 보유. 모호한 옵션은 임의 선택 대신 트리로 분기합니다.

에이전트가 분업

계획·작성·평가·보고를 별도 에이전트로 분리해, 한 세션이 모든 책임을 떠안지 않게 합니다.

Hook이 자동 검증

스펙 헤더 누락, 테스트 축 누락, 우회 패턴을 파일 작성 즉시 자동 검출합니다. 사람의 기억에 의존하지 않습니다.

Test Workflow

테스트 작업은 4개 에이전트가 순차로 협업합니다. 메인 세션은 직접 작성하지 않고 오케스트레이터 역할만 합니다.
