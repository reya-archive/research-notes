Microsoft 365 Copilot

Microsoft Graph + Semantic Index 기반 RAG가 내장된 M365 AI 어시스턴트. SharePoint/OneDrive 문서를 자동 인덱싱하며, Word/Excel/PowerPoint/Teams/Outlook에서 직접 사용할 수 있는 네이티브 통합이 핵심 강점입니다.

제품 개요

정식 명칭Microsoft 365 Copilot
제공사Microsoft
출시2023-11 GA (Enterprise), 지속 업데이트
카테고리Microsoft 관리형 SaaS 엔터프라이즈 AI 어시스턴트
핵심 가치M365 네이티브 통합 + SharePoint RAG 자동화 + Office 앱 내장 AI

아키텍처

핵심 구조 사용자 프롬프트 → Microsoft Graph + Semantic Index에서 관련 문서 검색(Grounding) → Azure OpenAI LLM 호출 → 후처리 → 응답 반환. 모든 데이터는 M365 서비스 경계 내부에서 처리됩니다.

계층컴포넌트 / 역할
오케스트레이터Copilot Orchestrator - 프롬프트 전처리, Grounding, LLM 라우팅, 후처리
검색 인덱스Semantic Index - 테넌트/사용자 레벨 벡터 인덱스 (Graph 콘텐츠 자동 인덱싱)
데이터 레이어Microsoft Graph - 메일, 채팅, 문서, 일정, 회의 등 RBAC 기반 접근
LLMAzure OpenAI Service (GPT 계열 - 최신 세대 기본 + 이전 세대 폴백)
확장Copilot Studio - 커스텀 에이전트, 플러그인, MCP 지원
거버넌스Entra ID 인증, Purview 민감도 레이블/DLP, 조건부 접근

주요 기능

RAG 엔진 / 문서 파싱

  • 파싱 엔진: Office 네이티브 파서. PPTX/XLSX/DOCX는 원본 포맷 그대로 이해
  • 지원 포맷: DOCX, PPTX, XLSX, PDF, TXT, HTML, MD, CSV, JSON, XML, 이미지, 코드 파일 등
  • PPTX/XLSX 파싱: Office 원본이므로 변환 손실 없음. 텍스트 박스, 표, 차트, 발표자 노트, SmartArt 텍스트 모두 추출
  • 이미지 이해: 2025년 말부터 vision 모델로 문서 내 임베디드 이미지 분석 가능 (OCR)
  • 문서 요약: 최대 약 150만 단어 (약 3,000페이지) 요약 가능

특수 파일 형식

  • 코드 파일: JS, Python, Java, C#, SQL 등 20여 종 직접 지원
  • Loop/Fluid: Microsoft Loop 컴포넌트 네이티브 지원
  • 이미지: PNG, JPG, GIF, BMP, TIFF - OCR + vision 분석
  • 오디오: Teams 회의 녹음 자동 전사 · 요약

LLM / 검색 / 권한

  • LLM 선택
    GPT 계열 (기본/폴백) + Claude Sonnet 계열 (Researcher)
  • 임베딩
    Semantic Index (비공개, text-embedding-3 계열 추정)
  • 컨텍스트
    GPT 계열 기준 (공식 수치 미공개, 일반적으로 128K~)
  • 검색 방식
    시맨틱 + 키워드 + RBAC 권한 인식
  • 폴더 구조
    SharePoint/OneDrive 구조 그대로 반영
  • SSO
    Entra ID (Azure AD) / SAML / 조건부 접근

커넥터 / 확장

  • 기본 데이터 소스: SharePoint Online, OneDrive, Teams, Exchange (Outlook/일정), OneNote, Loop
  • Copilot 커넥터: 100+ 사전 구축 (Confluence, Salesforce, ServiceNow, Jira, SAP 등)
  • Copilot Studio: 커스텀 에이전트 빌더. LLM 모델 직접 선택 가능 (GPT 계열, Claude 계열 등)
  • Declarative Agents: M365 오케스트레이터 활용 + 커스텀 지식/액션 정의
  • MCP 지원: Model Context Protocol로 외부 비즈니스 시스템 통합

가격 정책 (2026-04)

MCNC 권장: Business 플랜 MCNC는 100명 이하 + SI 프로젝트 50명 이하이므로 Copilot Business ($21/user/mo)로 충분합니다. Business와 Enterprise는 Copilot 엔진과 RAG 능력이 100% 동일하며, 사용자 한도(300명 vs 무제한)와 일부 거버넌스 기능(Purview DSPM 등 E5 전용)에서만 차이가 납니다.

추가 과금

  • 기본 Copilot 기능 (Word/Excel/PPT/Outlook/Teams 내 AI): 좌석 라이선스에 포함, 토큰 과금 없음
  • Copilot Studio 에이전트 메시지: 종량제 $0.01/message
  • Pay-as-you-go: Copilot Chat + SharePoint agents를 라이선스 약정 없이 종량제 사용 가능

리전 (한국 저장 가능, LLM 처리는 미지원)

  • 데이터 저장: 한국 리전 포함 (Product Terms 데이터 레지던시 커밋)
  • LLM 처리: 한국 in-country 처리 아직 미지원 (2026년 확대 예정 국가에 미포함)
  • Advanced Data Residency(ADR) 추가 구매 시 더 강력한 레지던시 보장 가능

데이터 처리 · 보안

  • Microsoft 약관: 프롬프트 · 응답 · Graph 데이터를 기반 LLM 학습에 사용하지 않음
  • 처리 데이터는 M365 서비스 경계 내부에서 전송 중 · 저장 시 암호화 (BitLocker, TLS, IPsec)
  • 접근 제어: Entra ID + RBAC + 조건부 접근 + MFA. Copilot도 사용자 권한 범위 내에서만 접근
  • 데이터 보호: Microsoft Purview 민감도 레이블 · DLP 정책 존중
  • 인증: ISO 27001/27018, ISO 42001 (AI 관리), SOC 2 Type II, GDPR, HIPAA, FedRAMP High
  • 데이터 상주: 한국 리전 데이터 저장 가능. LLM 호출은 다른 리전으로 라우팅될 수 있음

강점과 약점

강점 5

  • M365 네이티브: Word/Excel/PPT/Outlook/Teams에서 바로 AI 사용. 별도 UI 학습 불필요
  • Office 문서 파싱 최고: PPTX/XLSX를 원본 포맷으로 직접 이해, 변환 손실 없음
  • SharePoint RAG 자동화: 기존 SharePoint에 저장된 문서를 추가 설정 없이 즉시 검색
  • 100+ 커넥터: Confluence, Salesforce, Jira 등 외부 시스템 연동 가능
  • 한국 데이터 상주: 데이터 저장 시 한국 리전 사용 가능

약점 5

  • 이중 라이선스: M365 기본 + Copilot 추가. Business Standard+Copilot Business 시 $33.50/user/mo (Enterprise 조합은 $66)
  • 트라이얼 없음: 엔터프라이즈 무료 평가판 미제공, 연약정 필수
  • LLM 선택 제한: 기본 Copilot에서 모델 직접 선택 불가 (Copilot Studio 별도 구매 필요)
  • MCNC AWS 노하우 재사용 불가: Microsoft 생태계 의존
  • LLM 처리 리전: 한국 in-country LLM 처리 아직 미지원

MCNC · SI 적합도

  • 권장 라이선스: Copilot Business ($21) + Business Standard ($12.50). MCNC 100명 이하, SI 프로젝트 50명 이하라 Business 한도(300명) 충분. RAG 능력은 Enterprise와 100% 동일
  • 기존 노하우 활용도: 낮음. MCNC는 AWS 중심이라 MS 생태계 경험이 제한적
  • 반복 배포: M365 테넌트 단위 관리. 고객사마다 별도 테넌트 또는 사이트 컬렉션 분리 필요
  • SM 이관: 관리형이라 운영 부담 낮음. 단, M365 관리자 역량 필요
  • 프로젝트 간 이식성: 높음 (M365 사용 고객사 한정). 비M365 환경에는 부적합
