Gemini Enterprise + Google Drive

Gemini Pro 계열 LLM + Vertex AI Search + NotebookLM Enterprise + Agent Builder가 결합된 완제품. 1M~2M 토큰 컨텍스트, Document AI 최상위 파싱, Google Drive 네이티브 연동이 강점입니다.

제품 개요

정식 명칭Gemini Enterprise
제공사Google Cloud
출시2025 말 GA (Google Agentspace의 후신 · 리브랜딩)
카테고리Google 관리형 SaaS 엔터프라이즈 AI 에이전트 플랫폼
핵심 가치1M 토큰 컨텍스트 + Document AI 최상위 파싱 + Workspace 네이티브

제품 계층 구조

중요 유사 제품 간 혼동이 잦습니다. 결정권자 자료에서 반드시 구분을 명시하세요.

계층제품 / 역할
플랫폼 (완제품)Gemini Enterprise - 사용자 대상 엔터프라이즈 AI 어시스턴트 (Agentspace 후신)
런타임 / 빌더Vertex AI Agent Builder - 개발자 대상 에이전트 SDK/플랫폼
RAG 엔진Vertex AI Search - Gemini Enterprise 하위 검색 엔진
리서치 UINotebookLM Enterprise - Gemini Enterprise 번들 포함
Workspace 번들NotebookLM Plus - 별도 제품. Gemini Enterprise NotebookLM과 혼동 금지
소비자 플랜Google AI Pro / Ultra - 개인용 (본 조사 범위 밖)

6개 핵심 컴포넌트

  1. Gemini 모델: Gemini Pro 계열 (1M 토큰, 최대 64K 출력), Gemini Flash 계열 등 - 세대 주기적 갱신
  2. Agent Workbench: 에이전트 설계 · 오케스트레이션 UI
  3. 사전 제작 에이전트: Deep Research, NotebookLM, Idea Generation 등 즉시 사용 가능
  4. 데이터 커넥터: 100+ 엔터프라이즈 앱 연결 (Google Drive, Workspace, Vertex AI Search, Apigee 등)
  5. Agent Gallery: 사내 에이전트 공유 · 큐레이션
  6. 거버넌스: 권한 인식 검색, 감사 로그, 데이터 상주

주요 기능

RAG 엔진 / 문서 파싱

  • 파싱 엔진: Google Document AI (기업 문서 특화). 레이아웃 · 표 · 도형 인식 최강
  • 지원 포맷: PDF, DOCX, PPTX, XLSX, TXT, HTML, MD, CSV, JSON, 이미지 (OCR), 오디오, 비디오
  • PPTX/XLSX 파싱: 3사 중 최상위. 복잡 기획서 · 화면설계서 · 표 많은 Excel에서 우위
  • 1M 토큰 컨텍스트로 대용량 문서 통째로 이해 가능
  • 청킹/리랭킹: 자동 최적화 (관리형)

특수 파일 형식

  • 오디오 (m4a): 네이티브 지원. 음성 → 텍스트 자동 전사 (한국어 포함 다국어)
  • 다이어그램 (drawio, mmd): 이미지 파싱 경로. PNG 사전 변환 시 라벨 추출
  • XML 테이블: 구조 이해 3사 중 가장 우수 (1M 컨텍스트 + Gemini 구조 이해)

LLM / 검색 / 권한

  • LLM 선택
    Gemini Pro 계열 / Flash 계열
  • 임베딩
    gemini-embedding 계열 (관리형)
  • 컨텍스트
    1M ~ 2M 토큰
  • 검색 방식
    시맨틱 + 키워드 + 권한 인식
  • 폴더 구조
    Drive 구조 그대로 반영
  • SSO
    Workspace SSO / SAML

Google Drive Cataloging

Gemini Enterprise에서는 정식 GA 상태(2025년 말). Vertex AI Search 단독 사용 시에는 Preview + Allowlist 필요했으나, Gemini Enterprise 번들에서는 제약이 해제되어 Drive 폴더를 Data Store로 등록하면 권한 인식 상태에서 검색이 가능합니다.

  • Vertex AI 직접 호출 시에는 별도 토큰 단가 (입력 $1.25 / 출력 $5 per 1M, Gemini Pro 계열 기준)

데이터 처리 · 보안

  • Google 약관: 고객 데이터를 모델 학습에 사용하지 않음 (기본 설정)
  • 처리 데이터는 조직이 지정한 리전에 저장, 전송 중 · 저장 중 암호화
  • 암호화: 기본 Google 관리 키. CMEK (Standard/Plus) 지원
  • VPC Service Controls: 지원
  • 인증: ISO 27001/27017/27018, SOC 1/2/3, HIPAA, FedRAMP High (미국 리전), PCI DSS
  • 데이터 상주: 도쿄 리전 기준 일본 내 저장. "한국 내 저장" 요구 시 부적합

강점과 약점

강점 5

  • Document AI 파싱 품질 최상위: PPTX/XLSX 비중이 높은 데이터에 가장 강력
  • 1M~2M 토큰 컨텍스트: 폴더 전체 한 번에 이해
  • NotebookLM 내장: 영업팀 친숙한 UX 즉시 사용 (소비자 제품 인지도)
  • Drive 네이티브: 복잡한 폴더 트리 구조 그대로 보존
  • Deep Research 에이전트: 내부 문서 자동 리서치, 기획서 초안 작성

약점 5

  • 최고가: 50명×6개월 Business ₩945만, Plus ₩2,250만
  • 리전 제약: 한국 리전 없음 (도쿄/싱가포르)
  • MCNC 기존 노하우 활용 제한 (AWS 중심)
  • SSO 유연성: Workspace 외는 에디션 · 협의 필요
  • 벤더 락인: Google 생태계 의존

MCNC · SI 적합도

  • 기존 노하우 활용도: 낮음~중간. Workspace 쓰는 고객사 제안 시 시너지 큼
  • 반복 배포: Google Cloud 프로젝트 분리 + Workspace 조직 설정. Self-serve 한계, 영업 담당 배정 필요
  • SM 이관: 관리형이라 운영 부담 낮음. 단, Google 협의 채널 유지 필요
  • 프로젝트 간 이식성: 높음. 단 에디션 · 볼륨 할인 재협의 필요

주의사항

  • NotebookLM Plus ≠ Gemini Enterprise NotebookLM: 전자는 Workspace 추가 기능, 후자는 Gemini Enterprise 내장. 결정권자 자료에서 구분 필수
  • Agentspace 용어: 2025년까지 사용. 2026년 현재 공식명은 "Gemini Enterprise"
  • Gemini Pro 계열 최신 세대: 정기적으로 갱신되며 에이전트 과제 특화 개선이 누적됨 (도입 시점에 최신 세대 확인 권장)
