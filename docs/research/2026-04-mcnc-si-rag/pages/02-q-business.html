Amazon Q Business

AWS 가 운영하는 관리형 엔터프라이즈 RAG 서비스입니다. SharePoint · Confluence · Teams 등 기존 시스템 40가지 이상을 자동으로 연결할 수 있고, 60일 무료 트라이얼(50명) 로 비용 부담 없이 검증할 수 있습니다.

영업팀 핵심 3줄 ① SharePoint · Confluence 등 기존 시스템 자동 연결이 가장 큰 강점입니다 (40가지 이상). ② 데이터는 한국이 아닌 호주(시드니) 리전 에 저장됩니다 - 일반 민간사 SI 는 통상 가능하나 규제 산업(금융 · 의료 · 정부) 은 사전 검토 필요. ③ 60일 50명 무료 트라이얼 가능, 본운영은 1인당 월 $3 ~ $20.

시드니 리전 사용 - 영업 시 사전 안내 사항

Amazon Q Business 는 2026년 4월 기준 한국(서울) 리전에서 제공되지 않아, MCNC 는 가장 가까운 호주 시드니 리전 을 사용합니다. 이로 인해 다음 사항을 영업 단계에서 고객사에 안내해야 합니다.

  • 데이터 저장 위치 - 업로드 문서와 검색 인덱스는 호주에 저장됩니다. 일반 민간 기업 SI 는 통상 수용 가능하지만, 규제 산업(금융 · 의료 · 정부) 고객은 사전 동의 필수입니다.
  • 응답 속도 - 시드니와 한국 간 왕복 시간이 약 130 ~ 160ms 추가됩니다. 대화에서 체감은 가능하지만 사용 불가 수준은 아닙니다.
  • 일부 고급 기능 미지원 - 사용자 정의 앱 빌더, 외부 시스템 자동 호출 같은 고급 기능 일부는 시드니 리전에서 미지원입니다. 한국 고객 대다수는 영향 없습니다.
기술팀 참고 - 시드니 리전 지원/미지원 기능 상세

시드니에서 지원

  • 엔터프라이즈 RAG 질의응답
  • 파일 업로드 (PDF, 이미지 포함)
  • Tabular Search (소규모 표)
  • 스캔 PDF OCR
  • 임베디드 이미지 포함 파일
  • 콘텐츠 생성 (요약, 초안)

시드니에서 미지원

  • Q Apps (사용자 지정 앱 빌더)
  • Q Actions (외부 시스템 호출 에이전트)
  • Audio/Video 파일 파싱 (m4a 등)
  • Q in QuickSight 일부 기능

주요 기능

문서 자동 인식

  • 지원 형식 - PDF (스캔본 포함), Word, PowerPoint, Excel, 텍스트, HTML, CSV, JSON 등
  • 파워포인트 - 슬라이드별 텍스트와 삽입 이미지의 글자까지 자동 인식. 도형 레이아웃은 일부 손실
  • 엑셀 - 시드니 리전은 단순 표만 인식. 복잡한 셀 결합이 있는 표는 시트별로 분리 후 업로드 권장
  • 4개 후보 중 중상위 수준의 자동 인식 품질

특수 파일 처리

  • 음성 파일 (m4a) - 시드니 리전 미지원 (미국 리전만 지원)
  • 다이어그램 (drawio, mmd 등) - 직접 인식 불가. 이미지로 변환 후 OCR 경로
  • XML 표 - 표 구조 자동 인식 어려움. 엑셀로 변환 권장

AI 모델 · 검색 · 권한

  • AI 모델 선택
    불가 (AWS 내부 모델 고정)
  • 검색 방식
    키워드 + 의미 검색 혼합 (관리형)
  • 한 번에 읽는 문서량
    비공개
  • 문서 정리 단위

  • 통합 로그인
    AWS 통합 계정 시스템 연동 필수

사용자 권한 자동 반영

Q Business 는 고객사 계정 시스템과 연동해 사용자별 접근 권한을 자동으로 반영합니다. 예를 들어 영업팀 사용자가 재무팀 폴더 문서를 못 보는 상태라면, Q Business 답변에서도 해당 내용이 제외됩니다. MCNC SI 관점에서 고객사 조직 보안 모델을 그대로 활용할 수 있다는 점이 강점입니다.

기술팀 참고 - IAM Identity Center 연동 상세

Q Business 는 IAM Identity Center 필수 연동입니다. 소스 시스템(SharePoint, Confluence 등) 의 사용자별 ACL 을 상속해 답변에서 사용자 권한 범위 밖 문서를 자동 필터링합니다. 세부 연동은 AWS 공식 가이드 참고.

업로드 화면 (영업 시연 시 중요)

기본 업로드 화면은 AWS 콘솔 형태라 영업 시연에 부적합합니다. 현장 시연용으로는 다음 3가지 옵션 중에서 고객사 환경에 맞춰 선택합니다.

  1. 별도 업로드 화면 제작 - 드래그&드롭 가능한 간단한 업로드 페이지를 직접 만들고, 업로드된 파일을 Q Business 가 자동 인식하도록 구성
  2. SharePoint · Confluence 연동 - 고객사가 이미 사용 중인 시스템에 문서를 올리면 Q Business 가 자동으로 인식 (가장 자연스러운 시나리오)
  3. 채팅 화면 첨부 업로드 - 기본 채팅 화면(Web Experience) 의 첨부 기능 사용. 단, 영구 저장 아닌 한 번의 대화 내에서만 사용

인덱스 (시간당 - 검색 데이터 저장 비용)

Starter Index$0.14/시간 (월 환산 약 $102) - 단일 가용영역, 최대 20,000 문서 또는 200MB, 5 유닛 한도
Enterprise Index$0.264/시간 (월 환산 약 $193) - 3개 가용영역, 자체 암호화 키, 확장성 높음

권장 인덱스 Starter 200MB 한도는 일반 문서 수백 개면 충분하지만, 이미지 많은 디자인 파일은 용량이 빠르게 소진됩니다. 안전 마진과 자체 암호화 키 적용을 위해 본운영은 Enterprise Index 1 유닛 권장.

무료 트라이얼

  • 60일 무료, 최대 50명, Lite/Pro 어느 쪽이든 가능
  • 인덱스 1,500시간 무료 (60일 기준 Starter 1 유닛에 해당)
  • 프로토타입 검증 전 기간을 트라이얼 안에서 해결 가능

월 비용 시뮬레이션

사용자 구독료 + 인덱스 비용 합산. 환율 1 USD = 1,500 KRW 기준.

  • Lite + Starter (10명, 시연용)
    월 약 ₩158,000
    $30 + $75 = $105 / 월
  • Lite + Enterprise (50명, 본운영)
    월 약 ₩515,000
    $150 + $193 = $343 / 월
  • Pro + Enterprise (50명, 고급 기능)
    월 약 ₩1,790,000
    $1,000 + $193 = $1,193 / 월

영업 활용 포인트 60일 트라이얼 안에 프로토타입을 끝내면 구독료 $0. 본운영은 50명 기준 월 약 150만원(Lite + Enterprise) 수준으로 안내 가능.

연결 가능 시스템 (40가지 이상)

주요 시스템은 다음과 같으며, 전체 목록은 AWS 공식 문서를 참고하세요.

  • 문서 · 위키 - SharePoint · Confluence · Teams · OneDrive · Google Drive · Box · Dropbox
  • 고객 관리 · 티켓 - Salesforce · ServiceNow · Jira · Zendesk
  • 커뮤니케이션 - Slack · Gmail · Google Calendar
  • 저장소 · 개발 - Amazon S3 · GitHub 등
  • 그 외 30종 이상 지원

데이터 처리 · 보안

  • 데이터 학습 미사용 - AWS 약관상 고객 데이터를 모델 학습에 사용하지 않음. 입출력 별도 저장 없음 (Q Business 인덱스 내 저장 제외)
  • 암호화 - 기본 AWS 관리 키. Enterprise Index 는 고객 자체 암호화 키 사용 가능
  • 네트워크 격리 - VPC 전용 통로(VPC Endpoint) 지원으로 인터넷을 거치지 않고 호출 가능
  • 국제 보안 인증 - ISO 27001/27017/27018, SOC 1/2/3, HIPAA BAA, PCI DSS 등 보유

강점과 약점

강점 5

  • SharePoint · Confluence 등 40가지 이상 시스템 자동 연결
  • 고객사 사용자 권한을 그대로 반영해 답변에서 자동 필터링
  • 서버 운영 · 업데이트 · 백업을 AWS 가 모두 담당 (관리형)
  • 60일 50명 무료 트라이얼로 비용 부담 없는 검증
  • AWS 주변 인프라 경험 (계정 관리 · S3 · 네트워크 격리) 그대로 활용 가능

약점 5

  • 기본 업로드 화면이 AWS 콘솔 형태라 영업 시연에 부적합 (별도 화면 또는 SharePoint 연동 필요)
  • 시드니 리전 일부 고급 기능 제약 (사용자 정의 앱, 음성 파일 등)
  • AI 모델이 AWS 내부 모델로 고정 - Claude 또는 GPT 로 교체 불가
  • Pro 요금은 1인당 월 $20 - 50명 × 6개월 환산 시 약 900만원, 기능 활용도 사전 검토 필요
  • 데이터가 한국이 아닌 호주에 저장 - 규제 산업 고객은 사전 동의 필수

MCNC · SI 적합도

  • 반복 배포 - 프로젝트별로 별도 애플리케이션 단위로 분리. CDK 템플릿화 시 수 시간 내 셋업
  • SM 이관 - 관리형이라 운영 부담 낮음. 콘솔 권한과 연결 설정만 인계
  • 프로젝트 간 이식성 - 높음. CDK 템플릿만 재사용
