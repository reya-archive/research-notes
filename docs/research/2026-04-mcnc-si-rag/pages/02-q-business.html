Amazon Q Business

AWS가 운영하는 엔터프라이즈 검색 · 생성형 AI. 40+ 커넥터와 IAM Identity Center 권한 자동 상속이 강점인 관리형 RAG. 60일 50명 무료 트라이얼로 프로토타입을 $0로 검증할 수 있습니다.

제품 개요

정식 명칭Amazon Q Business
제공사AWS (Amazon Web Services)
출시2024-05 GA (시드니 리전 2025-03 출시)
카테고리AWS 관리형 SaaS 엔터프라이즈 RAG 어시스턴트
핵심 가치40+ 커넥터로 기존 시스템 즉시 연결 + IAM 권한 자동 상속 + 관리형 운영

시드니 리전 사용 배경

  • 서울 리전 미지원: 2026-04 기준 서울(ap-northeast-2)에서 Amazon Q Business 제공 안 됨
  • 아시아 퍼시픽 최근접: 시드니(ap-southeast-2)가 MCNC 관점 현실적 선택
  • 지연 시간: 시드니 ↔ 서울 왕복 약 130~160ms. 대화형 UX에서 체감 가능하나 치명적 수준은 아님
  • 데이터 상주: 호주 내 저장. "한국 내 보관" 요구 고객사에는 부적합 - 민간 일반 SI에서는 수용 가능한 경우가 많음

시드니에서 지원

  • 엔터프라이즈 RAG 질의응답
  • 파일 업로드 (PDF, 이미지 포함)
  • Tabular Search (소규모 표)
  • 스캔 PDF OCR
  • 임베디드 이미지 포함 파일
  • 콘텐츠 생성 (요약, 초안)

시드니에서 미지원

  • Q Apps (사용자 지정 앱 빌더)
  • Q Actions (외부 시스템 호출 에이전트)
  • Audio/Video 파일 파싱 (m4a 등)
  • Q in QuickSight 일부 기능

주요 기능

RAG 엔진 / 문서 파싱

  • 파싱 엔진: Amazon Textract + 자체 파이프라인. 스캔 PDF OCR, 표 구조 인식 지원
  • 지원 포맷: PDF (스캔 포함), DOCX, PPTX, XLSX, TXT, HTML, MD, CSV, JSON 등
  • PPTX 파싱: 슬라이드별 텍스트 + 임베디드 이미지 OCR. 도형 레이아웃 일부 손실
  • XLSX 파싱: 표 구조 인식. 단, 시드니 리전은 "작은 표"만 지원
  • 3사 중 중상위. Document AI(Gemini) 대비 열세, Dify 대비 우위

특수 파일 형식

  • 오디오 (m4a): 시드니 리전 미지원. 미국 리전만 지원
  • 다이어그램 (drawio, mmd): 네이티브 파싱 없음. PNG/SVG 변환 후 OCR 경로
  • XML 테이블: 표 구조 자동 인식 어려움. XLSX 변환 권장

IAM Identity Center 권한 자동 상속

Q Business는 IAM Identity Center 필수 연동. 소스 시스템(SharePoint, Confluence 등)의 사용자별 ACL을 상속해 답변에서 사용자 권한 범위 밖 문서를 자동 필터링합니다. 예: 영업팀 사용자가 재무팀 폴더 문서를 못 보는 상태라면 Q Business 답변에서도 해당 내용이 제외됩니다. MCNC SI 관점에서 고객사 조직 보안 모델을 그대로 반영할 수 있다는 점이 강점입니다.

업로드 UX (중요)

기본 경로는 Admin 콘솔 업로드 + S3 커넥터 중심 설계이므로 영업팀이 AWS 콘솔로 직접 업로드하는 것은 UX가 나쁩니다. 대안:

  1. S3 버킷을 Dify 네이티브 UI 또는 자체 FE로 추상화. 업로드 → S3 → Q Business 자동 크롤링
  2. Web Experience(내장 챗 UI)의 첨부 업로드 활용. 단, 영구 인덱싱 아닌 대화 컨텍스트 한정
  3. SharePoint/Drive 등 이미 쓰는 시스템을 커넥터로 연결 (영업팀 기존 도구 유지)

가격 정책 (2026-04)

사용자 구독 (월)

Lite$3 / user / mo - 기본 Q&A, 권한 반영 응답
Pro$20 / user / mo - 전체 기능 (Q Apps, Q in QuickSight Reader Pro 등. 시드니는 일부 제한)
중복 할당Lite + Pro 둘 다 할당 시 Pro 요금만 부과

인덱스 (시간당)

Starter Index$0.14 / hr / unit (단일 AZ, 최대 20k 문서 or 200MB, 5 유닛 한도)
Enterprise Index$0.264 / hr / unit (3개 AZ, CMK 암호화, 확장성 높음)
월 환산Starter 1 유닛 $102 / Enterprise 1 유닛 $193

참고 샘플(591파일 · 0.9GB) 기준 적합성 Starter Index 200MB 한도는 추출 텍스트 기준이라 PPTX+XLSX 위주 데이터는 Starter로도 가능성 있음. 안전 마진 확보 및 CMK 암호화를 위해 Enterprise Index 1 유닛 권장.

무료 트라이얼

  • 60일 무료, 최대 50명, Lite 또는 Pro 어느 쪽이든
  • 인덱스 1,500시간 무료 (60일 기준 Starter 1 유닛 ≈ 1,440시간 상응, 충분)
  • 프로토타입 전 기간 트라이얼 내에서 해결 가능

40+ 커넥터 (대표)

  • 콘텐츠/위키: Confluence(Cloud/Server), SharePoint(Cloud/Server), Teams, Exchange, OneDrive, Google Drive, Box, Dropbox
  • CRM/티켓: Salesforce, ServiceNow, Jira, Zendesk
  • 커뮤니케이션: Slack, Gmail, Google Calendar (Preview)
  • 저장소: Amazon S3, Amazon FSx Windows, Smartsheet
  • 개발: GitHub (Cloud/Server)
  • 커스텀: Custom Data Source Connector (API), Web Crawler

데이터 처리 · 보안

  • AWS 약관: 고객 데이터를 모델 학습에 사용하지 않음. 입출력 데이터 저장 없음 (Q Business 인덱스 내 저장 제외)
  • 암호화: 기본 AWS 관리 키. Enterprise Index는 CMK 지원
  • VPC Endpoint: 지원. Private 트래픽 가능
  • 인증: ISO 27001/27017/27018, SOC 1/2/3, HIPAA BAA, PCI DSS, FedRAMP Moderate (미국 리전)

강점과 약점

강점 5

  • 40+ 커넥터로 고객사 기존 시스템 즉시 연결
  • IAM Identity Center + 소스 ACL 자동 상속
  • 관리형 운영: 패치 · 스케일링 · 백업 AWS 책임
  • 60일 50명 무료 트라이얼로 프로토타입 $0
  • AWS 주변 인프라 경험 활용 가능 (IAM · S3 · VPC 수준)

약점 5

  • 업로드 UX: S3/Admin 콘솔이라 영업팀 시연 어려움
  • 시드니 리전 기능 제약: Q Apps/Actions, Audio/Video 미지원
  • LLM 선택 불가: AWS 내부 모델, 벤더 락인 강함
  • Pro 요금: 50명×6개월 ₩900만 (과투자 리스크)
  • 한국 외 데이터 상주 (시드니)

MCNC · SI 적합도

  • 기존 노하우 활용도: 제한적. RAG 엔진은 AWS 관리형이라 MCNC가 직접 개입 불가. AWS 주변 인프라 운영 경험(IAM · S3 · VPC · CDK)만 셋업 단계에 활용 가능
  • 반복 배포: Q Business Application 단위로 프로젝트별 분리. CDK 템플릿화 시 수 시간 내 셋업
  • SM 이관: 관리형이라 운영 부담 낮음. 콘솔 권한과 커넥터 설정만 인계
  • 프로젝트 간 이식성: 높음. CDK 템플릿만 재사용
