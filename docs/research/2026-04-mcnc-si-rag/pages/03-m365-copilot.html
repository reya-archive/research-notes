Microsoft 365 Copilot

Microsoft 가 제공하는 M365 통합 AI 어시스턴트입니다. SharePoint · OneDrive · Teams 문서를 별도 업로드 없이 자동 인식하고, Word · Excel · PowerPoint 안에서 직접 AI 기능을 호출할 수 있습니다.

영업팀 핵심 3줄 ① 별도 업로드 단계가 없습니다 - SharePoint/OneDrive/Teams 문서를 자동 인식. ② Office 앱에서 직접 사용 가능해 직원 교육 부담이 가장 적습니다. ③ 비용은 이중 라이선스 구조(M365 기본 + Copilot 추가) 라 1인당 월 $33.50 수준 - 고객사가 이미 M365 사용 중이어야 합리적.

아키텍처

사용자가 질문하면 Copilot 이 SharePoint · OneDrive · Teams 문서를 자동 검색해 답변 근거로 사용하고, GPT 계열 AI 엔진으로 답변을 생성합니다. 모든 데이터는 Microsoft 365 안에서만 처리되며 외부로 나가지 않습니다.

기존 SharePoint 의 사용자별 접근 권한을 그대로 반영하므로, 영업팀 사용자가 재무팀 폴더 문서를 못 보는 상태라면 Copilot 답변에서도 자동으로 제외됩니다.

주요 기능

문서 자동 인식

  • 지원 형식 - Word · Excel · PowerPoint · PDF · 텍스트 · HTML · 이미지 · 코드 파일 등 일반 문서 대부분
  • Office 문서를 원본 형식 그대로 직접 인식 - 텍스트 박스, 표, 차트, 발표자 노트, SmartArt 텍스트까지 변환 손실 없이 추출
  • 이미지 분석 - 2025년 말부터 문서 안의 이미지도 자동 인식 (글자 추출 + 내용 분석)
  • 긴 문서 요약 - 최대 약 3,000페이지 분량까지 한 번에 요약 가능
  • 음성 회의 - Teams 회의 녹음을 자동 텍스트화하고 요약
참고 - 지원 특수 파일 형식 상세
  • 코드 파일 - JS, Python, Java, C#, SQL 등 20여 종 직접 지원
  • Loop · Fluid - Microsoft 신규 협업 문서 형식 네이티브 지원
  • 이미지 - PNG, JPG, GIF, BMP, TIFF - 글자 인식과 내용 분석 모두 지원
  • 오디오 - Teams 회의 녹음 자동 전사 · 요약

AI 모델 · 검색 · 권한

  • AI 모델 선택
    불가 (GPT 계열 고정)
  • 한 번에 읽는 문서량
    GPT 계열 기준 (공식 비공개, 통상 128K 이상)
  • 검색 방식
    의미 + 키워드 + 사용자 권한 인식
  • 문서 정리 단위
    SharePoint/OneDrive 폴더 구조 그대로 반영
  • 통합 로그인
    기존 Microsoft 계정 그대로 사용

외부 시스템 연결 · 확장

  • 기본 인식 데이터 - SharePoint · OneDrive · Teams · Outlook · 일정 · OneNote · Loop (Microsoft 자체 시스템 전부)
  • 외부 시스템 연결 - 100가지 이상 사전 구축 (Confluence · Salesforce · ServiceNow · Jira · SAP 등)
  • 커스텀 에이전트 - Copilot Studio 로 직접 제작 가능 (별도 라이선스 필요, 대부분 프로젝트는 불필요)
참고 - 고급 기능 (대부분 프로젝트는 불필요)
  • Copilot Studio - 커스텀 에이전트 빌더. AI 모델 직접 선택 가능 (GPT 계열, Claude 계열 등). $200/테넌트/월 추가
  • Declarative Agents - M365 오케스트레이터 활용 + 커스텀 지식/액션 정의
  • MCP 지원 - Model Context Protocol 로 외부 비즈니스 시스템 통합

가격 (2026-04 기준)

MCNC 권장 조합 MCNC 는 100명 이하 + SI 프로젝트 50명 이하이므로 Copilot Business ($21) + M365 Business Standard ($12.50) = 1인당 월 $33.50 조합을 권장합니다. Business 와 Enterprise 는 Copilot AI 엔진과 RAG 능력이 동일하며, 사용자 한도(300명 vs 무제한) 와 일부 고급 보안 기능에서만 차이가 납니다.

Copilot 라이선스 (1인당 월)

Copilot Business ★ 권장$21 - 연 약정. 300명 한도. M365 Business Standard/Premium 등 기본 라이선스 필요
Copilot Enterprise$30 - 연 약정. 무제한. M365 E3/E5 기본 라이선스 필요
Copilot Chat$0 - M365 구독에 포함. 웹 채팅만 가능, RAG 불가
Copilot Studio (선택)$200/테넌트/월 - 커스텀 에이전트 빌더 (대부분 불필요)

M365 기본 라이선스 (Copilot 사용에 필수)

Business Basic$6 - Copilot Business 자격 충족 최소
Business Standard ★ MCNC 권장$12.50 - Office 앱 데스크톱 포함
Business Premium$22 - 추가 보안 기능 강화

최종 비용 = 기본 라이선스 + Copilot. MCNC 권장 조합 (Business Standard + Copilot Business) 은 1인당 월 $33.50 수준으로, Enterprise 조합 (E3 + Copilot Enterprise = 약 $66) 의 절반입니다.

리전 (데이터 저장 vs AI 처리)

  • 데이터 저장 위치 - 한국 리전 사용 가능 (계약서로 보장)
  • AI 처리 위치 - 한국 리전 미지원, 다른 리전(미국 등) 으로 라우팅됨. 일반 민간사 SI 는 통상 가능, 규제 산업은 사전 검토 필요
  • 고급 데이터 상주 보장이 필요하면 Advanced Data Residency 추가 구매 가능

월 비용 시뮬레이션

Copilot 추가 라이선스 기준. 환율 1 USD = 1,500 KRW 기준.

  • Business 연약정 (10명, 시연용)
    월 약 ₩315,000
    $21 × 10 = $210 / 월. M365 기본 라이선스 별도 ($125 추가 시 약 ₩503,000)
  • Business 연약정 (30명)
    월 약 ₩945,000
    $21 × 30 = $630 / 월. M365 기본 포함 시 약 ₩1,508,000
  • Business 연약정 (50명, 본운영)
    월 약 ₩1,575,000
    $21 × 50 = $1,050 / 월. M365 기본 포함 시 약 ₩2,513,000

프로토타입 비용 안내 엔터프라이즈 무료 트라이얼이 없어 1년 약정 ($21 × 10명 × 12개월 = $2,520/년) 이 필요합니다. 대안으로 종량제 (Pay-as-you-go) 로 Copilot Chat + SharePoint Agents 만 일시 테스트 가능. 이미 M365 Business Standard 등을 보유한 고객사라면 기본 라이선스 비용은 추가 발생하지 않습니다.

데이터 처리 · 보안

  • 데이터 학습 미사용 - Microsoft 약관상 프롬프트 · 응답 · 문서 데이터를 AI 학습에 사용하지 않음
  • 처리 경로 - 모든 데이터는 M365 서비스 경계 내부에서만 처리되며 전송 · 저장 시 암호화 (BitLocker, TLS 등)
  • 접근 제어 - 기존 Microsoft 계정 + 권한 + 다단계 인증 그대로 적용. Copilot 도 사용자 권한 범위 내에서만 문서 접근
  • 민감 정보 자동 보호 - Microsoft Purview 의 민감도 레이블 · 데이터 유출 방지(DLP) 정책 자동 반영
  • 국제 보안 인증 - ISO 27001/27018, ISO 42001 (AI 관리), SOC 2 Type II, GDPR, HIPAA 등 보유
  • 데이터 저장 위치 - 한국 리전 보관 가능. AI 처리 자체는 다른 리전으로 갈 수 있음

강점과 약점

강점 5

  • 별도 업로드 단계 없음 - SharePoint · OneDrive · Teams 문서를 자동 인식
  • Office 문서 인식 품질 최고 - PPTX/XLSX 를 원본 형식 그대로 이해, 변환 손실 없음
  • Word · Excel · PPT 안에서 직접 AI 사용 - 직원 교육 부담 최소화
  • Confluence · Salesforce · Jira 등 100가지 이상 외부 시스템 연결 가능
  • 데이터 저장 위치를 한국 리전으로 지정 가능

약점 5

  • 이중 라이선스 - M365 기본 + Copilot 추가 필요. MCNC 권장 조합도 1인당 월 $33.50
  • 무료 트라이얼 없음 - 엔터프라이즈 평가판 미제공, 1년 약정 필수
  • AI 모델이 GPT 계열로 고정 - Claude 등 다른 모델로 교체 불가 (Copilot Studio 별도 구매 시만 가능)
  • MCNC AWS 운영 노하우 재사용 어려움 - Microsoft 생태계 중심
  • AI 처리 자체가 한국 리전 미지원 - 일반 민간사는 통상 가능, 규제 산업은 사전 검토 필요

MCNC · SI 적합도

  • 권장 라이선스 - Copilot Business ($21) + M365 Business Standard ($12.50). MCNC 는 100명 이하, SI 프로젝트 50명 이하라 Business 한도(300명) 충분. AI 능력은 Enterprise 와 동일
  • 반복 배포 - M365 테넌트 단위로 관리. 고객사마다 별도 테넌트 또는 SharePoint 사이트 분리 필요
  • SM 이관 - 관리형이라 운영 부담 낮음. 단, M365 관리자 역량 필요
  • 프로젝트 간 이식성 - M365 사용 고객사 한정으로 높음. 비M365 환경에는 부적합
