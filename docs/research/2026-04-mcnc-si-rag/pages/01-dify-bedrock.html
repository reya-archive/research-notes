Dify + Amazon Bedrock

셀프호스팅 LLMOps(Dify) + Bedrock LLM 의 2-Tier 조합. 데이터 주권과 저비용이 강점이며, MCNC 기존 노하우(Docker · PGVector · Bedrock)를 100% 활용할 수 있습니다.

제품 개요

정식 명칭Dify (langgenius/dify)
제공사LangGenius (2023 출시, 2026-03 $30M 투자 유치) + AWS
카테고리OSS 셀프호스팅 LLMOps + Amazon Bedrock
라이선스Dify Apache 2.0 (Community Edition 전 기능)
핵심 가치셀프호스팅 데이터 주권 + 최신 Claude 모델 호출 + Dify 네이티브 업로드 UI

  • Dify: 업로드 UI · RAG 관리 · 워크플로우 · API 게이트웨이. LLM 자체는 호스팅하지 않음

주요 기능

RAG 엔진 / 문서 파싱

  • Knowledge Pipeline (2025~2026 도입): OCR → 파싱 → 구조화 추출 → 청킹 → 임베딩 비주얼 파이프라인
  • 지원 포맷: PDF, DOCX, PPTX, XLSX, MD, HTML, TXT, CSV, JSON 등
  • PPTX/XLSX 파싱: 기본 ETL은 텍스트 추출 위주, Unstructured ETL 연동 시 레이아웃 인식 향상. 3사 중 중간 수준 (Document AI 대비 열세, Q Business 대비 유사)
  • 청킹: 문자 기반 · 시맨틱 · 부모-자식 계층형 선택 가능
  • 리랭커 교체 가능 (BGE-Reranker, Cohere Rerank 등)
  • 네이티브 업로드 UI: 드래그&드롭, 다중 파일, 진행률, 즉시 인제스트

특수 파일 형식

  • 오디오 (m4a): 네이티브 transcription 없음. Whisper 플러그인 또는 외부 변환
  • 다이어그램 (drawio, mmd): 텍스트만 추출, 시각 구조 손실. PNG/SVG 사전 변환 권장
  • XML 테이블: 텍스트로 읽히나 구조화 인식은 제한적. Excel 변환이 실용적

LLM / 임베딩 / 기타

가격 정책 (2026-04)

Dify 본체

  • Community Edition (셀프호스팅): 무료, Apache 2.0, 전 기능 사용 가능
  • Dify Cloud(참고, 본 검증 미사용): Sandbox Free / Pro $59 / Team $159 / Enterprise 별도

Amazon Bedrock (토큰 단가, 도쿄 리전 기준)

Claude Sonnet 계열입력 $3 / 출력 $15 (per 1M 토큰, 현행 세대 기준)
Titan Embeddings v2$0.02 / 1M 토큰
Batch Inference50% 할인
Prompt CachingWrite 1.25x, Read 0.1x

권장 인프라 사양 (고객사 환경 별도)

  • EC2 t3.xlarge (4 vCPU / 16 GB RAM): Dify Docker Compose 최소 권장. API + Worker + Sandbox 컨테이너 동시 구동
  • EBS gp3 100 GB: Weaviate / PostgreSQL 볼륨 + 업로드 파일 캐시
  • 네트워크: Bedrock 호출용 VPC endpoint 또는 NAT Gateway

인프라 비용은 고객사 환경에 따라 결정 사내 가상화 플랫폼·기존 EC2 계정·공용 컨테이너 런타임 등 재사용 여부에 따라 실비용이 크게 달라집니다. 본 자료는 단가 산정 대신 권장 스펙만 제시합니다.

데이터 처리 · 보안

  • 데이터 주권: 셀프호스팅이므로 모든 문서가 MCNC 인프라 내부에 머무름. 외부 전송은 Bedrock LLM API 호출 시만
  • Bedrock 약관: 고객 데이터를 모델 학습에 사용하지 않음, 입출력 데이터 저장 없음
  • 암호화: Bedrock KMS + EBS 암호화
  • VPC: 모든 컴포넌트 Private Subnet 배치 가능. Bedrock은 VPC Endpoint 지원
  • SSO: Community Edition 미지원 (Enterprise Edition만). 약점
  • 인증: Bedrock ISO 27001, SOC 1/2/3, HIPAA BAA

강점과 약점

강점 5

  • 데이터 주권: 모든 문서가 MCNC 인프라 내부에 머무름
  • 저비용: 50명×6개월 ₩180만~240만 (4사 중 최저)
  • MCNC 노하우 직결: mcnc-rag, Docker, PGVector, Bedrock 경험 100% 활용
  • 커스터마이징 자유도: UI · 워크플로우 · 사내 연동 제약 없음
  • LLM 선택 자유: Claude Sonnet · Haiku · Opus 교체 가능, 벤더 락인 없음

약점 5

  • SSO 부재: Community Edition은 SAML/OIDC 미지원
  • 문서 파싱: Document AI 대비 복잡 기획서에서 열세
  • 폴더 구조: 복잡한 폴더 트리 그대로 표현 어려움 (평면 + 태그)
  • 운영 책임 전가: 패치 · 백업 · 모니터링 MCNC 책임
  • 검색 품질 튜닝 공수: 리랭커 · 청킹 전략 조정이 MCNC 책임

MCNC · SI 적합도

  • 기존 노하우 활용도: 매우 높음. mcnc-rag (Open WebUI + vLLM), Docker Compose, PGVector, Bedrock CDK 자동화 모두 그대로 투입
  • 반복 배포: Docker Compose + Bedrock 계정 분리를 Terraform/CDK화 시 수 시간 내 신규 프로젝트 셋업
  • SM 이관: 매뉴얼 · Runbook 인계 가능. 단, MCNC 기술팀이 운영 지식 구축 필요
  • 프로젝트 간 이식성: 매우 높음. 구성 전체가 사내 자산
