MCNC SI RAG 시스템 프로토타입 검증 결과

Open WebUI + Bedrock / Amazon Q Business / Microsoft 365 Copilot / Claude Teams - 4개 후보 제품을 동일 기준으로 비교한 내부 검증 자료입니다. 영업팀이 제품을 선택할 수 있도록 기술 팩트와 조건별 가이드를 제공합니다.

이 자료 한 줄 요약 MCNC 기술팀이 4개 후보 제품의 프로토타입을 구축 · 검증한 결과를 7개 카테고리(가격 · 기능 · 보안 · 시연 등)로 비교한 발표 자료입니다. 단일 최종 권장안은 포함하지 않으며, 조건별 선택 가이드만 제공합니다.

    1인당 월 비용 약 $1 ~ $3 Claude Sonnet 4.6 토큰 단가 기준 · 인프라 별도 (EC2 t3.large ~ m6i.xlarge)

    사내 보관 · 저비용 · LLM 선택 자유가 우선일 때

    PPTX/XLSX 파싱
    ★★★★★
    업로드 UX
    ★★★★★
    LLM 자유 (Claude · Nova · OSS) 구축 중간 (Docker Compose)
    RAG 시스템 - 직접 설치형 · Claude 구동
    강점
    • 모든 문서가 회사 서버 안에만 머무름 (외부 전송 없음)
    • AWS 의 Claude 모델을 그대로 사용 - 답변 품질 최상위권
    • 프로젝트 특성에 맞춰 답변 품질을 세밀하게 조정 가능
    • 라이선스 비용 0원, AWS 사용량(토큰)만 월 과금
    • 향후 다른 AI 모델(Nova · 국산 LLM 등)로 쉽게 교체 가능
    약점
    • 서버 · 업데이트 · 백업을 MCNC 가 직접 운영
    • 복잡한 파워포인트/엑셀 표는 마이크로소프트 · 구글 대비 다소 아쉬움
    • 대량 문서를 한꺼번에 올릴 때 AWS 측 처리 속도 제한이 있음
    • 영업팀이 직접 설정 변경 불가, 관리자 화면에서 세팅 후 배포

    PersistentConfig 함정 · Bedrock Titan 쿼터 · 운영은 MCNC 책임

  • AWS 관리형 SaaS

    Amazon Q Business

    40+ 커넥터와 IAM 권한 자동 상속이 강점인 관리형 엔터프라이즈 RAG

    1인당 월 비용 $20 Pro 기준 · Lite $3 · 인덱스 비용 별도

    기존 시스템 연동 · 관리형 운영 · 권한 자동 상속이 우선일 때

    PPTX/XLSX 파싱
    ★★★★★
    업로드 UX
    ★★★★★
    LLM 불가 (AWS 내부) 구축 낮음 (AWS 콘솔)
    RAG 시스템 - AWS 완전 관리형
    강점
    • 서버 운영 · 업데이트 · 백업을 AWS 가 전부 담당
    • SharePoint · Confluence · 드롭박스 등 고객사 시스템 40개 이상 기본 연결 지원
    • 기존 AWS 사용자 계정 체계로 "누가 어떤 문서 볼 수 있는지" 자동 반영
    • 60일 무료 체험(50명) 으로 부담 없이 검증 가능
    • AWS 의 문서 추출 엔진이 표 · 서식을 안정적으로 읽음
    약점
    • 답변 품질을 세밀하게 조정할 수 있는 옵션이 제한적
    • 서울이 아닌 호주 시드니 서버 사용 - 데이터 상주 요구 고객사는 사전 검토 필요
    • 파일 업로드 화면이 개발자용 콘솔 형태라 영업 시연에 부적합
    • AI 모델이 AWS 내부 모델로 고정 (Claude · GPT 교체 불가)

    데이터 상주는 시드니. 영업팀 시연용 업로드 UI 추상화 필요

    Microsoft 365 Copilot

    1인당 월 비용 $21 Business 플랜 (300명 한도) · M365 Business Standard 별도 ($12.50)

    M365 생태계 · Office 내장 AI · 교육 부담 최소가 우선일 때

    PPTX/XLSX 파싱
    ★★★★★
    업로드 UX
    ★★★★★
    LLM 불가 (GPT 계열 고정) 구축 낮음 (M365 관리센터)
    RAG 시스템 - Microsoft 365 내장형
    강점
    • 별도 업로드 단계 자체가 없음 - SharePoint · OneDrive · Teams 문서 자동 인식
    • 기존 MS 계정 체계로 "누가 어떤 문서 볼 수 있는지" 자동 적용
    • 워드 · 엑셀 · 파워포인트 원본을 변환 없이 직접 읽음 (표 · 서식 손실 최소)
    • 기존 오피스 앱 안에서 AI 기능이 바로 제공 (직원 교육 부담 최소)
    약점
    • 외부 시스템(AWS · 사내 기간계 DB 등) 연결은 별도 개발 필요
    • 답변 품질 문제 발생 시 내부 조정이 어려움 (내부 동작 비공개)
    • AI 모델이 GPT 계열로 고정 - Claude 교체 불가
    • 1인당 월 $21 연약정 필수, 무료 체험 없음

    이중 라이선스 (Business+Copilot $33.50/user/mo) · 엔터프라이즈 트라이얼 없음

    Claude Teams

    Anthropic 직접 구독 + Projects 기능을 RAG처럼 활용하는 가장 단순한 도입 옵션

    1인당 월 비용 $20 Team Standard 연간 결제 ($25 월간), 최소 5석

    인프라 0 · 빠른 도입 · 최신 모델 자동 업데이트가 우선일 때

    PPTX/XLSX 파싱
    ★★★★★
    업로드 UX
    ★★★★★
    LLM Claude 한정 (Sonnet · Haiku · Opus) 구축 최저 (시트 구매만)
    RAG 시스템 - Projects 기능으로 대체
    강점
    • 서버 설치·운영 작업 자체가 없음 - Anthropic 콘솔에서 시트만 구매하면 즉시 사용
    • 프로젝트별로 지식 파일과 커스텀 지침을 묶어 간이 RAG 형태로 활용
    • Claude 최신 세대(Sonnet 4.6 · Opus 4.x) 자동 업데이트 - 답변 품질 최상위권

    약점

    • 커스텀 커넥터 개발 불가 (기본 Google Drive · GitHub 외 확장 제한)
    • MCNC AWS · Docker 노하우 재사용 불가 (Anthropic 직접 구독)

    무료 트라이얼 없음 · HIPAA · 감사 로그 · SCIM 등 고급 기능은 Enterprise 필요
