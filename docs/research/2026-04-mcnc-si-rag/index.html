참고 샘플 데이터 (기존 SI 프로젝트 1건)

MCNC 전체 데이터가 아니라 기존에 진행했던 SI 프로젝트 1건을 예시로 골라 분석한 결과입니다. 이번 조사·비교·시연은 이 샘플을 기준으로 진행했으며, PPTX + XLSX가 63%를 차지해 Office 문서 파싱 품질을 핵심 평가 기준으로 삼았습니다. 실제 프로젝트마다 파일 수 · 용량 · 형식 비율은 크게 달라질 수 있습니다.

배경과 요구사항

MCNC는 일반 민간 기업 대상 모바일 · 웹 SI(System Integration) 사업을 수행합니다. 신규 SI 프로젝트에 반복 투입할 사내 LLM + RAG 시스템을 선정하기 위한 조사 · 비교 · 시연용 자료입니다.

  • 규모 총 등록 사용자 약 50명, 동시 사용자 20~30명 (초기 분석/설계 피크)
  • 운영 모델 프로젝트마다 독립된 경량 인스턴스를 새로 구축. 고객사 전체 공유 시스템 아님
  • 제약 조건 클라우드 기반, 학습 미사용 약관, 계약비 내 저비용, 반복 배포 가능, SM 이관 가능
